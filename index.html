Our Main Aim is

Interact with Bluetooth devices using Web Browser


Before we start:

Just think that you can interact with your BLE devices with your Web Broswers without installing any S/W (Zero FootPrint). Yes its possible with Web Bluetooth APi provided by Chrome Browser.
Your basic knowledge on Bluetooth Low Energy (BLE) and the Generic Attribute Profile (GATT) is enough to achive this.

Chrome Bluetooth API is on Beta mode and many enthusiastic developers are supporting the same to give a complete package.

    A subset of the Web Bluetooth API is available from Chrome 56 for Chrome OS, Chrome for Android M, and Mac,Windows. This means you should be able to
  • request and connect to nearby Bluetooth devices,
  • read/write Bluetooth characteristics,
  • receive GATT Notifications,
  • know when a Bluetooth device gets disconnected,and even
  • read and write to Bluetooth descriptors.

POC:

It is a small possbility check that how to interact this Web Bluetooth Api with BLE enabled device(smart phone, Glucometer, Fitness tracker). The web browsers are enabled with Web Bluetooth package and a smart BLE device(glucometer, fitness band or smart phone) are connected through BLE protocol. Then this browser will connect to the GATT server of the target device. Since its a smart device, it should have pre defined services running. Through code we are going to use this service, such a way that information on the device will be transfered to the browser.

Bluetooth Low Energy (BLE):

Bluetooth Low Energy is a wireless personal area network technology developed for healthcare, fitness, beacons, security, and home entertainment industries.

BLE is used in smart medical devices to track on the usage of the device and hence it directly helps the patient health.

Why BLE? It consumes least amount of battery and works with high efficency.

People with chronic medical conditions, like diabetes, it’s very important to take regular blood glucose readings, and keep accurate notes. Patients must log what their blood sugar level is, when they take insulin and the amount. Low-cost BLE devices in this dosing device and in the instrument will do all this logging for the patient, and, with consent, share this information with the doctor. This could revolutionize the quality of such a patient’s healthcare management.
